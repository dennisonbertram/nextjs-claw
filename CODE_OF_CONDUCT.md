# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good faith.
- Critique code, not people.
- No harassment, slurs, or personal attacks. Doesn't matter who.
- Disagreements happen. Resolve them with reasoning, not heat.

## Reporting

If someone violates the code of conduct, file an issue or email the maintainer (see `package.json` `author` field). We'll act promptly.

## Enforcement

Maintainers reserve the right to remove comments, commits, code, issues, and other contributions not aligned with this Code of Conduct, and to ban contributors for behaviors they deem inappropriate.

## Scope

Applies inside this repo (issues, PRs, discussions, commits) and any other project space (chat, conferences) where you're representing the project.
